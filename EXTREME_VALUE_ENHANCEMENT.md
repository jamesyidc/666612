## CSS 动画

### pulse 动画（图标脉动）
```css
@keyframes pulse {
    0%, 100% { 
        transform: scale(1);
        opacity: 1;
    }
    50% { 
        transform: scale(1.1);
        opacity: 0.8;
    }
}
```
- 周期：2秒
- 效果：图标放大缩小 + 透明度变化
- 第二个卡片延迟0.5秒，形成交错效果

### glow 动画（发光效果）
```css
@keyframes glow {
    0%, 100% {
        box-shadow: 0 8px 24px rgba(239, 68, 68, 0.3);
    }
    50% {
        box-shadow: 0 8px 32px rgba(239, 68, 68, 0.6), 
                    0 0 20px rgba(239, 68, 68, 0.4);
    }
}
```
- 周期：3秒
- 效果：阴影强度周期性增强，形成呼吸灯效果
